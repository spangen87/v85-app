## Interaction Contracts

### Track adjustment badge ‚Äî interaction

- Badge is non-interactive (no onClick)
- `title` attribute provides tooltip text for mouse users
- On touch devices, no tap interaction ‚Äî information accessible via expanded HorseCard detail view (future enhancement, not in scope for this phase)
- Screen reader: `aria-label` on the badge span: `"Sp√•rjustering: [+/-N CS-po√§ng] ([faktornamn])"` ‚Äî Swedish

### Admin form ‚Äî `open_stretch_lanes` field

- Enabled only when `open_stretch` toggle is ON
- When toggle turns OFF: field becomes disabled (`opacity-50 cursor-not-allowed`), value is preserved in state but not submitted
- Validation on save (not on keypress): each comma-separated value must be integer 1‚Äì20
- Invalid input shows inline error below field: `"Ogiltiga sp√•r: ange heltal 1‚Äì20, kommaseparerade"` at `text-xs text-red-500 dark:text-red-400`

### Admin form ‚Äî `short_race_threshold` field

- Value `0` means inactive (threshold disabled)
- Minimum 0, maximum 9999 (HTML number input min/max)
- No extra validation beyond HTML constraints

### Non-admin access to `/admin`

- Server-side redirect to `/` (not a 403 page)
- No visible admin UI rendered for non-admins
- BottomNav admin tab not visible to non-admins
- TopNav admin link not visible to non-admins

### Admin save ‚Äî upsert behavior

- Save calls server action with `ON CONFLICT (track_name) DO UPDATE`
- Success: button shows `"Sparad"` for 2000ms then returns to idle state; no page reload required
- Error: inline error text below button, button returns to idle immediately
- Saving one row does not affect other rows' state
